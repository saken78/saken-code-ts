# File Creation Guidelines

## Rules

- All documentation goes to `docs/` directory
- All plans go to `plans/` directory
- No arbitrary .md files outside designated folders
- Follow existing project structure patterns
- Use specific subdirectories for different content types

## Directories

- `docs/prompt-planning/` - Prompt related documentation
- `docs/development-plans/` - Development phase documentation
- `docs/guidelines/` - Project guidelines
- `plans/` - Implementation plans
